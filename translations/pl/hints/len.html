Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

string w Pythonie to lista znaków lub symboli. Tak więc ciąg "Hi" jest listą dwóch elementów "H" i "i". Tak więc prawie wszystkie funkcje listy działają poprawnie z ciągiem.

Pana celem jest przekonwertowanie liczby całkowitej value na ciąg, a następnie znalezienie długości tego ciągu (lub znalezienie długości listy znaków).

Proszę mi wskazać?

len() to funkcja do znajdowania długości, a str() - do konwersji obiektu na ciąg znaków.

Ok. Jak mogę ich użyć?

Proszę najpierw przekonwertować podaną liczbę całkowitą na ciąg znaków. 

text = str(value)

Jak użyć drugiej funkcji?

Proszę teraz znaleźć długość text. 

text = str(value)       
length = len(text)

Jak będzie wyglądało moje ostateczne rozwiązanie?

Musi Pan tylko zawinąć kod w wyrażenie funkcji i zwrócić length.

Oto całe rozwiązanie:

def number_length(value: int) -> int:

    text = str(value)
    length = len(text)

    return length

Można to też zrobić w jednej linii:

def number_length(value: int) -> int:

    return len(str(value))

Jasne, proszę łapać!

Proszę rekurencyjnie wykonać dzielenie liczb całkowitych i policzyć każdy poziom rekurencji, aż value stanie się 0. 

def number_length(value: int) -> int:

    return 1 + ((value:= value // 10) and number_length(value))

Proszę zrobić to samo, co w poprzednim rozwiązaniu, ale jaśniej. Dopóki value nie ma więcej niż 1 cyfry (> 9), proszę dodać 1 do digits i wykonać dzielenie całkowite. Dlaczego digits ma wartość początkową 1? Ponieważ liczby nieujemne mniejsze niż 10 (nawet 0) mają długość 1 cyfry. 

def number_length(value: int) -> int:

    digits = 1
    while value> 9:
        digits += 1
        value //= 10
        
    return digits
