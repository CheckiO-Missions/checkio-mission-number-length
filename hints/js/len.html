You just need to wrap the code into function expression and return the len.

Here is the whole solution:

import assert from "assert";


function numberLength(value: number): number {
    
    const text = String(value);
    const len = text.length;
    
    return len;
}

Recursively make integer division and count each level of recursion until value become 0. 

import assert from "assert";


function numberLength(value: number): number {

    return value/10 >= 1 ? numberLength(value/10)+1 : 1;
}

Do the same as in previous solution, but clearer. Until the reminder of division value by powered 10 not equal value, add 1 to digits. Why digits has initial value 1? To get non-zero reminder, you have to divide at least by 10. When the reminder becomes equal, it means you count all digits in value. 

import assert from "assert";


function numberLength(value: number): number {
    
    var digits = 1;
    while ( (value % (10**digits)) !== value )
        digits++;

    return digits;
}
