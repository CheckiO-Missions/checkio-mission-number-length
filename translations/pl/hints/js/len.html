Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

pana celem jest przekształcenie liczby value w żądło, a następnie znalezienie długości tej struny.

Jakich instrumentów szukam?

Potrzebuje Pan metody String() do konwersji liczby na ciąg i właściwości string.length.

Jak mogę ich użyć?

Najpierw przekonwertujmy podaną liczbę value na ciąg znaków. 

const text = String(value);

Jak użyć drugiego instrumentu?

Proszę teraz znaleźć długość text. 

const text = String(value);
const len = text.length;

Jak będzie wyglądało moje ostateczne rozwiązanie?

Musi Pan tylko zawinąć kod w wyrażenie funkcji i zwrócić len.

Oto całe rozwiązanie:

function numberLength(value: number): number {
    
    const text = String(value);
    const len = text.length;
    
    return len;
}

Można to też zrobić w jednej linii:

function numberLength(value: number): number {
    
    return String(value).length
}

Jasne, proszę łapać!

Proszę rekurencyjnie wykonać dzielenie liczb całkowitych i policzyć każdy poziom rekurencji, aż value stanie się 0. 

function numberLength(value: number): number {

    return value/10>= 1? numberLength(value/10)+1: 1;
}

Proszę zrobić to samo, co w poprzednim rozwiązaniu, ale jaśniej. Dopóki przypomnienie o dzieleniu value przez zasilane 10 nie będzie równe value, proszę dodać 1 do digits. Dlaczego digits ma wartość początkową 1? Aby uzyskać niezerowe przypomnienie, należy podzielić co najmniej przez 10. Gdy przypomnienie stanie się równe, oznacza to, że policzył Pan wszystkie cyfry w value. 

function numberLength(value: number): number {
    
    var digits = 1;
    while ( (value % (10**digits))!== value )
        digits++;

    return digits;
}
