You just need to wrap the code into function expression and return the length.

Here is the whole solution:

def number_length(value: int) -> int:

    text = str(value)
    length = len(text)

    return length

def number_length(value: int) -> int:

    return len(str(value))

Recursively make integer division and count each level of recursion until value become 0. 

def number_length(value: int) -> int:

    return 1 + ((value := value // 10) and number_length(value))

Do the same as in previous solution, but clearer. Until the value has more then 1 digit (> 9) add 1 to digits and make integer division. Why digits has initial value 1? Because non-negative numbers less then 10 (even 0) have 1 digit length. 

def number_length(value: int) -> int:

    digits = 1
    while value > 9:
        digits += 1
        value //= 10
        
    return digits
