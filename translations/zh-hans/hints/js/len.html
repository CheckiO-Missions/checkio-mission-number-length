你只需要把代码包装成函数表达式并返回len 。

下面是整个解决方案：

function numberLength(value: number): number {
    
    const text = String(value);
    const len = text.length;
    
    return len;
}

或者你可以在一行中完成：

function numberLength(value: number): number {
    
    return String(value).length
}

递归地进行整数除法，并计算每一级递归，直到value 成为0。 

function numberLength(value: number): number {

    return value/10>= 1? numberLength(value/10)+1: 1;
}

做法与前面的解决方案相同，但更清晰。10 直到除数value 的提示值不等于value ，在digits 上加 1。为什么digits 的初始值是1 ？为了得到非零的提醒，你必须至少除以10。当提醒变得相等时，这意味着你计算了value 中的所有数字。 

function numberLength(value: number): number {
    
    var digits = 1;
    while ( (value % (10**digits))!== value )
        digits++;

    return digits;
}
