Твоя мета — перетворити ціле число на рядок, а потім знайти довжину цього рядка.

Які інструменти я шукаю?

Тобі потрібен рядковий метод String(), щоб конвертувати число в рядок, і атрибут string.length.

Добре, як мені ними скористатися?

Давай спочатку ти перетвориш дане ціле число у рядок. 

const text = String(value);
А як використати атрибут?

Тепер знайди довжину text. 

const text = String(value);
const len = text.length;
То як має виглядати моє остаточне рішення?

Тобі лише потрібно обгорнути код у вираз функції і повернути len.

Ось повне рішення:

import assert from "assert";


function numberLength(value: number): number {
    
    const text = String(value);
    const len = text.length;
    
    return len;
}

Або це можна записати в один рядок:

import assert from "assert";


function numberLength(value: number): number {
    
    return String(value).length
}

Звісно, тримай!

Рекурсивно здійснюй цілочисельне ділення і рахуй рівень рекурсії поки value не стане 0. 

import assert from "assert";


function numberLength(value: number): number {

    return value/10 >= 1 ? numberLength(value/10)+1 : 1;
}

Роби те ж, що і в минулому розв'язку, але зрозуміліше. Допоки остача від ділення value на 10 у степені не рівний value, додавай 1 до digits. Чому digits мають початкове значення 1? Щоб отримати не-нульову остачу, ти має поділити щонайменше на 10. Коли остача стане рівною числу, це означає, що пораховано всі цифри value. 

import assert from "assert";


function numberLength(value: number): number {
    
    var digits = 1;
    while ( (value % (10**digits)) !== value )
        digits++;

    return digits;
}
